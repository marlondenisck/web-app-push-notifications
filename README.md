Explorando notificações push na prática 🚀

Neste exemplo com a Web Push API, mostro um fluxo completo funcionando de ponta a ponta:

🔔 Ao abrir o navegador, o usuário recebe o pedido de permissão
📋 Em outro navegador, a inscrição aparece na dashboard em tempo real
📨 Envio uma notificação e ela chega instantaneamente
🔒 Mesmo com o navegador fechado, a notificação é recebida
🌐 Ao clicar nela, o navegador abre automaticamente na página — mostrando o poder das notificações offline


Simples, direto ao ponto e focado no que realmente importa: funcionamento real.
